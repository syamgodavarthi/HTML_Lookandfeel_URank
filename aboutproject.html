About Project

In this project we are providing an exclusive search Engine for University. Where we can find various aspects for requested University. Below is the flow of our project.

1) User can search for any specific University (or) Can directly search for any university in the University score card tab

2) Then the Specific University page will be redirected. Information about the specific University will be displayed on this page.

3) There is Learn about URank reference link, Which explains the data source and procedure followed for Ranking the University
